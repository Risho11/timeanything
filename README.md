# Time Anything

Free online timers and clocks at **[timeanything.com](https://www.timeanything.com)**.

## Tools

- [HIIT & Tabata Interval Timer](https://www.timeanything.com/interval-timer.html) — custom work/rest/prep time, audio cues, Tabata preset (20s/10s/8 rounds)
- [Pomodoro Timer](https://www.timeanything.com/pomodoro.html) — fully customizable work, short break, and long break durations; background-accurate
- [Stopwatch with Lap Timer](https://www.timeanything.com/stopwatch.html) — hundredths of a second, record laps, copy laps to clipboard
- [Countdown Timer](https://www.timeanything.com/countdown-timer.html) — audio alert at zero, quick presets, meeting timer
- [Focus Timer](https://www.timeanything.com/focus-timer.html) — single deep work session with task name display; no forced breaks
- [Event Countdown](https://www.timeanything.com/countdown-to-date.html) — count down to any date in days, hours, minutes, seconds
- [Egg Timer](https://www.timeanything.com/egg-timer.html) — soft boiled, medium, hard boiled presets + audio alert
- [Online Clock](https://www.timeanything.com/online-clock.html) — live 12/24-hour time with date and day of week

No sign-up. No app. Everything runs in the browser.
