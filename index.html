About these tools

TimeAnything is a set of free, no-sign-up timer tools. All timers run entirely in your browser using JavaScript — no server, no data collection. Close the tab and the timer stops; that is the only caveat.

Each tool includes audio alerts using the Web Audio API so you do not need to watch the screen.
